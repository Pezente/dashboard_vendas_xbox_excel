# dashboard_vendas_xbox_excel

Dashboard em Excel de vendas de inscrições do Xbox.

Os dados utilizados possuem informações de vendas de planos de inscrição do Xbox Game Pass, EA Play Season Pass e Minecraft Season Pass.

O dashboard possui filtros para o período de assinatura (mensal, quadrimestral e anual), além de separação dos números totais pelo critério de a assinatura possuir ou não auto-renovação.
